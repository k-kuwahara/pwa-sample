# PWA Sample with Riot.js

## Usage

```bash
# preparing the environment
$ git clone git@github.com:kkeeth/pwa-sample.git

# install packages
$ yarn

# build
$ yarn build

# boot app
$ yarn server

# boot app & development
$ yarn start

# development
$ yarn watch
```
